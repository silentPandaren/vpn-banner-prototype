# VPN Banner — MY.GAMES Market

Прототип предупреждения о VPN. Показывает перетяжку «Использование VPN может приводить к неправильной работе сайта» с кнопкой «Понятно».

## Логика определения VPN

Используем два клиентских сигнала без сетевых запросов:

| Сигнал | API | Пример |
|--------|-----|--------|
| Язык браузера | `navigator.language` | `ru-RU`, `en-US` |
| Таймзона | `Intl.DateTimeFormat().resolvedOptions().timeZone` | `Europe/Moscow`, `America/New_York` |

### Правило

Баннер показывается, если **одновременно**:
1. Язык браузера — СНГ (`ru`, `be`, `uk`, `kk`, `ky`, `uz`, `tg`, `az`, `hy`, `ka`)
2. Таймзона — **не** из списка СНГ-зон

Логика: пользователь с русскоязычной системой и московским языком, но с европейской/американской таймзоной — скорее всего использует VPN, который меняет геолокацию, но не меняет настройки ОС.

### Почему именно эта пара

- **Только язык** — ненадёжно: у эмигрантов язык ru, но VPN не используется
- **Только таймзона** — ненадёжно: командировки, путешествия
- **Язык + таймзона** — пересечение двух независимых сигналов даёт меньше ложных срабатываний

### Список таймзон СНГ

Россия (16 зон), Беларусь, Украина, Казахстан (6 зон), Азербайджан, Армения, Грузия, Узбекистан, Туркменистан, Таджикистан, Кыргызстан, Молдова, Эстония, Латвия, Литва.

### Что не учитывается (намеренно)

- IP-геолокация — требует сетевого запроса, задержка, платные API
- `navigator.languages` (все языки) — избыточно, берём первый
- User-Agent, WebRTC leak — слишком сложно и ненадёжно

## Поведение баннера

- Появляется снизу с анимацией через 600 мс после загрузки
- Кнопка «Понятно» → закрывает + пишет флаг в `localStorage`
- После закрытия больше не показывается (до очистки localStorage)
